## 2. 技术描述
- 前端：React@18 + TypeScript + TailwindCSS@3 + Vite
- 初始化工具：vite-init
- 后端：Express@4 + TypeScript (ESM)
- 数据库：内存Mock数据（无需真实数据库，使用Mock数据模拟）
- 状态管理：Zustand
- 路由：React Router DOM
- 图表：Recharts
- 图标：Lucide React
- 二维码：qrcode.react

## 4. API 定义

### 4.1 认证
```typescript
// POST /api/auth/login
interface LoginRequest {
  username: string;
  password: string;
  role: 'visitor' | 'curator' | 'conservator' | 'security' | 'director';
}
interface LoginResponse {
  token: string;
  user: {
    id: string;
    name: string;
    role: string;
    avatar: string;
  };
}
```

